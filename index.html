Приклад використання списків визначень

HTML (HyperText Markup Language)
Мова розмітки гіпертексту, яка є основою будь-якої вебсторінки. Вона використовується для структурування контенту (тексту, зображень, посилань, таблиць тощо) за допомогою тегів, визначаючи ієрархію та взаємозв'язок елементів.
CSS (Cascading Style Sheet)
Мова таблиць каскадних стилів, яка відповідає за візуальне представлення вебсторінки. Вона контролює, як елементи HTML будуть виглядати: їхній колір, шрифти, розміри, розташування та адаптивність для різних пристроїв (дизайн і макет).
JavaScript
Скриптова мова програмування, яка додає інтерактивність та динамічну поведінку до вебсторінок. Вона дозволяє "оживити" статичний контент, обробляти події користувача (натискання кнопок, введення даних), взаємодіяти з сервером без перезавантаження сторінки тощо.
Front-end
Клієнтська частина вебсайту, з якою безпосередньо взаємодіє користувач через браузер. Включає всі елементи інтерфейсу, дизайн та інтерактивність, реалізовані за допомогою HTML, CSS та JavaScript.
